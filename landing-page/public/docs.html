REWIND_OPERATIONS // V2.0

CORE PROTOCOL: SESSION PRESERVATION

01 // GETTING STARTED

TO ACTIVATING THE KINETIC VOID TRACKING SYSTEM, FOLLOW THESE STEPS:

  • 01 INSTALL THE EXTENSION IN CHROME OR FIREFOX.
  • 02 PIN THE ICON TO YOUR TOOLBAR FOR INSTANT ACCESS.
  • 03 NAVIGATE TO ANY VIDEO STREAMING PLATFORM (YOUTUBE, NETFLIX, ETC.).
  • 04 THE SYSTEM WILL AUTOMATICALLY DETECT THE <video> ELEMENT.

02 // DASHBOARD CONTROLS

THE OPERATOR DASHBOARD IS YOUR COMMAND CENTER:

  • USER HISTORY: VIEW THE LAST 20 TRACKED CLUSTERS.
  • ANALYTICS: MONITOR YOUR CONSUMPTION DENSITY ACROSS 7 DAYS.
  • MANUAL ENTRY: OVERRIDE THE AUTO-PROBE TO LOG DATA MANUALLY.
  • REFRESH_SYNC: FORCE A FIREBASE UPLINK TO SYNC WITH REMOTE NODES.

03 // CLOUD SYNC

BY DEFAULT, ALL DATA IS LOCAL. TO ENABLE CROSS-DEVICE TRACKING:

  • CLICK LOGIN / SIGN UP ON THE LANDING PAGE.
  • AUTHENTICATE VIA THE SECURE GOOGLE UPLINK.
  • YOUR HISTORY WILL NOW BE SYNCED TO THE FIRESTORE CLOUD NODES.

04 // TROUBLESHOOTING

IF THE TRACKER FAILS TO INITIALIZE:

  • ENSURE THE VIDEO PLAYER IS USING HTML5 (FLASH IS NOT SUPPORTED).
  • GIVE THE PAGE A HARD REFRESH TO RE-INJECT THE CONTENT SCRIPT.
  • CHECK IF THE EXTENSION BADGE IS SHOWING AS ACTIVE.
